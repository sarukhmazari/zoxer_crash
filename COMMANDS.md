### Premium Access
- Some commands require premium access
- Premium can be added by owner using `addprem` command
- Premium users can use VIP bug commands and mode commands

### Number Format
- All commands that require phone numbers use format: `62xxx`
- Example: `6281234567890` (Indonesian number format)
- You can also mention users: `@username`

### Access Levels
- **Public**: Everyone can use
- **Premium**: Requires premium access
- **Owner**: Only bot owner can use
